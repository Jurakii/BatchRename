# Batch Rename
Super simple UI based batch renamer

## Setup
Download
Run the exe file or create a short cut to the exe

## Usage
1) Browse or Input a file path pointing to you files you wish to batch rename
2) The **Replace** field is the text you wish to replace
3) The **Name** field is you replacement text
4) Press **Rename**, if you miss a step or run into an error it will be displayed next to the button
5) If successful you will see a confirmation message
